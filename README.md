# ADHD 助手

专为 ADHD 人群设计的任务管理与专注辅助应用。

## 功能

- 🧩 **原子化拆解** - 将大任务分解为 ≤15 分钟的微任务
- 🎯 **单一焦点** - 一次只显示一个任务，减少干扰
- ⏱️ **视觉计时器** - 圆环动画展示时间流逝
- 📳 **体感时标** - 周期性震动提醒
- 🛡️ **专注结界** - 管理系统通知
- 💡 **思维中转站** - 语音/文字快速记录灵感

## 本地开发

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
```

## 技术栈

- React 18 + TypeScript
- Vite
- PWA (可安装到手机)
